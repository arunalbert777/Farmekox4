# **App Name**: FarmekoX

## Core Features:

- Phone Authentication: Simple phone number authentication for user registration and login.
- Dashboard: Centralized dashboard displaying key agricultural data.
- Smart Weather Card: 7-day weather forecast display with a refresh button for updated information.
- Crop Calendar: Interactive calendar showcasing sowing and irrigation tasks based on crop selection and local climate data.
- Language Toggle: Switch between English and Kannada languages for UI and AI audio.
- AI Chat and Voice Assistant: Gemini AI powered chat and voice assistant for crop recommendations and fertilizer identification using barcode photos, integrated with Firebase AI Logic SDK. The AI tool will use crop type, location, weather conditions, and identified fertilizer ingredients, in making a suggestion to the user.
- Interactive Maps: Integration of Google Maps API to display nearby seed stores and Mandi locations.
- Real-time Mandi Prices: Display of real-time local crop prices sourced from local market data.

## Style Guidelines:

- Primary color: Sage green (#A6C48A), inspired by the lush greenery of farmlands, creates a sense of growth and natural harmony.
- Background color: Clay white (#F5F3E6), resembling the earth, provides a clean, minimalist backdrop that allows the sage green elements to stand out.
- Accent color: Muted olive (#8A9A5B), complements the sage green, giving depth to the design.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look.
- Use minimalist, line-based icons in sage green and clay white to maintain a clean and consistent aesthetic.
- Employ a topographic design pattern in the background with subtle elevations, providing a tactile and high-end feel, similar to the reference images.
- Subtle, fluid transitions and animations to enhance user engagement without being distracting.
